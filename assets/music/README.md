# Música de Fondo

Coloca aquí 5-10 tracks de dark ambient / música misteriosa en formato .mp3, .wav u .ogg.

El sistema elige un track aleatorio para cada video y lo mezcla al 6% de volumen con fade in/out.

## Dónde descargar gratis (uso comercial permitido)

### Pixabay Music (recomendado, sin registro)
Buscar en: https://pixabay.com/music/
- Términos de búsqueda: `dark ambient`, `mystery`, `horror ambient`, `conspiracy`, `suspense`
- Filtrar por: "Free to use" (todos lo son en Pixabay)

### Nombres sugeridos para guardar los archivos
```
dark_ambient_01.mp3
dark_ambient_02.mp3
mystery_drone_01.mp3
suspense_loop_01.mp3
horror_ambient_01.mp3
```

### Otras fuentes
- **Free Music Archive**: https://freemusicarchive.org/ (filtrar por CC0 o CC-BY)
- **Incompetech**: https://incompetech.com/ (Kevin MacLeod, buscar "horror" o "mystery")
- **YouTube Audio Library**: https://studio.youtube.com/ → Audio Library → filtrar "Dark"

## Duración recomendada
- Mínimo 30 segundos (el sistema hace loop automático si el track es más corto que el video)
- Ideal: 1-3 minutos para variedad

## Nota
Sin música el sistema funciona igual — el video solo tendrá narración y SFX.
